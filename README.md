# PureScript Hello World (Web)

## Command

```
# Setup
$ npm run setup

# Run (localhost:8080)
$ npm run server
```

## Directories

- [src/Counter](./src/Counter) : カウンタ本体 (lib)
- [src/UI](./src/UI) : UI 制御 (exe)
